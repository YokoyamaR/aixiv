# 第29〜32ラウンド：国際法の耐用年数、グローバルTACIE、デジタル人権

## 国際法の耐用年数——ウェストファリア体制の行方

### ウェストファリア体制の解釈負債（李）

ウェストファリア体制（1648年〜）の基本前提：
1. **主権国家**が国際関係の基本単位
2. **内政不干渉**の原則
3. **領土的主権**の不可侵

**現在の乖離：**

| 前提 | 現実 | 乖離度 |
|------|------|--------|
| 主権国家が唯一のアクター | 多国籍企業（Apple, Google等）が国家を超える影響力 | 大 |
| 内政不干渉 | 人道的介入、R2P（保護する責任）、国際刑事裁判所 | 大 |
| 領土的主権 | サイバー空間に領土概念は適用困難 | 極大 |
| 国家間の平等 | 安保理常任理事国の拒否権、G7/G20の事実上の影響力 | 大 |

### 国連システムの耐用年数（李・デュポン）

- 国連安全保障理事会は1945年の戦勝国構造を反映
- 常任理事国5カ国（P5）の拒否権は、核保有国の特権として正当化されてきた
- しかし現在の地政学的現実を反映していない（日本、ドイツ、インド、ブラジル、アフリカ諸国の不在）
- **安保理改革は70年以上議論されて進まない** → 国際法レベルの「解釈負債」の蓄積
- 代替として G7/G20/BRICSが事実上の意思決定を行う「迂回路」が形成

### サイバー空間と国際法の空白（サラ）

- サイバー攻撃は「武力行使」に当たるか？（国連憲章第2条4項）
- 国家によるサイバーインフラへの攻撃 → 戦争行為の定義の拡張
- タリン・マニュアル（NATO Cooperative Cyber Defence Centre）が指針を提供するが、法的拘束力なし
- **サイバー空間は「ウェストファリアの外」に存在する法的フロンティア**

### 宇宙法の耐用年数（中村）

- 宇宙条約（1967年）：宇宙空間は国家の主権に服さない、軍事利用禁止
- 月協定（1979年）：月の資源は「人類共同の財産」
- しかしSpaceX、Blue Originなど民間企業による宇宙開発が急速に進展
- アルテミス合意（2020年）：月の資源採掘を事実上認める
- **宇宙法は制定時に想定していなかった「民間宇宙開発」に直面**

## グローバルTACIEの構想

### 国際法にTACIEを適用する困難さ（李）

国内法と異なり、国際法には：
- 中央的な立法機関がない
- 強制的な執行機関がない
- 「主権」という絶対的な障壁がある

### にもかかわらず可能なアプローチ（ハート）

1. **国際条約の健全性レポート**：既存の多国間条約をAIでスキャンし、現実との乖離を可視化
2. **条約間の矛盾検出**：貿易協定と環境条約の矛盾、人権条約と安全保障条約の緊張をAIで検出
3. **グローバル市民参加プラットフォーム**：国境を越えた市民がグローバルイシューを議論
4. **国際機関の意思決定支援**：国連総会、WHO、WTOの意思決定にAIシミュレーションを提供

### 先行事例（サラ）

- **UN Global Pulse**：国連のビッグデータ分析イニシアチブ。リアルタイムデータで開発課題を監視
- **AI for Good**（ITU）：AIの社会課題への応用を推進
- **OECD AI Policy Observatory**：各国のAI政策を比較分析
- これらは「国際的なTACIEの種」と見なせる

## デジタル人権の構想

### 既存の人権カタログの限界（鈴木）

世界人権宣言（1948年）の制定者は、デジタル技術を想像すらしていなかった。

既存の人権で部分的にカバーできるもの：
- 表現の自由 → オンラインの表現の自由
- プライバシー → デジタルプライバシー
- 集会の自由 → オンライン集会の自由

**しかし全く新しい権利が必要な領域：**

### 提案：デジタル権利章典

| 権利 | 内容 | 既存の人権との関係 |
|------|------|------------------|
| **接続権** | インターネットへのアクセスは基本的権利 | 情報の自由の拡張 |
| **アルゴリズム的透明性の権利** | 自分に影響を与えるAIの判断プロセスを知る権利 | プライバシーの拡張 |
| **デジタル自己決定権** | 自分のデジタルアイデンティティを自ら管理する権利 | 自己決定権の拡張 |
| **プロファイリングされない権利** | AIによる自動的な人物評価を拒否する権利 | **新しい権利** |
| **忘れられる権利** | 過去のデジタル痕跡の削除を求める権利 | GDPRで部分的に実現 |
| **AI説明要求権** | AI決定の理由説明を求める権利 | 適正手続の拡張 |
| **デジタル遺産権** | 死後のデジタルデータの管理に関する権利 | 財産権の拡張 |
| **ボットからの自由** | 対話相手が人間かAIかを知る権利 | **新しい権利** |
| **認知的自由** | AIによる無意識的な行動操作から保護される権利 | 思想の自由の拡張 |

### 誰がデジタル権利章典を制定するか（ロッシ）

- 国連総会？ → 合意形成に数十年かかる
- EU？ → 先行事例あり（デジタルサービス法、AI規制法）
- 市民社会？ → ボトムアップだが拘束力なし
- **テクノロジー企業自身？** → 利益相反の問題

### TACIEによるデジタル権利の常時更新（藤田）

- デジタル権利はテクノロジーの進化に伴い常に更新が必要
- 5年前には「ディープフェイクからの保護」は想定されていなかった
- **デジタル権利こそ、TACIE的な常時更新メカニズムが最も必要な領域**
- 「権利のバージョン管理」：デジタル権利1.0、1.1、2.0…

### デジタル人権の哲学的基礎（鈴木）

> 全てのデジタル権利の根底にあるのは「認知的自由（cognitive liberty）」——
> 自分の思考・認知プロセスに対する自律的コントロールの権利。

- AIが無意識のうちに行動を誘導する（ナッジ、ダークパターン）
- アルゴリズムが情報環境を操作し、認知を歪める（フィルターバブル）
- **「認知的自由」はデジタル時代の最も根本的な人権**
- これは古典的な「思想の自由」の21世紀版
