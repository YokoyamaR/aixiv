# 第57〜60ラウンド：戦争法のAI時代、身体と法、法のゲーミフィケーション

## 自律型兵器と国際人道法

### 国際人道法の基本原則と自律型兵器の衝突（李）

| 原則 | 内容 | 自律型兵器との問題 |
|------|------|-----------------|
| 区別原則 | 戦闘員と民間人を区別する義務 | AIが民間人を正確に識別できるか？ |
| 比例原則 | 軍事的利益と民間被害のバランス | AIに「比例性」の判断ができるか？ |
| 軍事的必要性 | 必要な範囲でのみ武力を行使 | AIの「必要性」判断の基準は？ |
| マルテンス条項 | 人道と公共の良心の命令に従う | AIに「良心」はない |

### 自律型兵器の規制動向（サラ）

- **CCW（特定通常兵器使用禁止制限条約）GGE**：2017年から議論しているが法的拘束力のある合意に至らず
- **Lethal Autonomous Weapons Systems (LAWS)** の定義すら合意できていない
- 米国・ロシア・中国が法的規制に消極的
- 30以上の国が予防的禁止を支持
- **AIの軍事利用は法の耐用年数問題の最も危険な側面**

### 「キルチェーン」の法的分析（鈴木）

攻撃決定の過程（キルチェーン）のどこに人間を介在させるか：
- **Human-in-the-loop**：人間が攻撃を承認する
- **Human-on-the-loop**：AIが攻撃し、人間が中止できる
- **Human-out-of-the-loop**：完全自律型

> 「人間が殺すことの道徳的責任」は法の根幹。AIが殺す場合、責任は誰にあるか。
> 開発者？指揮官？国家？AIそのもの？
> **「責任の空白」（accountability gap）**が国際人道法の最大の課題。

### TACIEと軍事法（デュポン）

- 軍事技術の進歩はつねに法を追い越してきた（毒ガス→化学兵器禁止条約、核兵器→NPT）
- しかしAI兵器の進化速度は過去の軍事技術と比較にならない
- 条約交渉に5年かかる間に、技術は3世代進む
- **軍事法こそTACIE的な「常時更新」が最も切迫して必要だが、最も導入困難な領域**

## 法と身体——人間の身体の法的境界の揺らぎ

### 身体の法的概念の拡張（田中）

| 技術 | 法的問い |
|------|---------|
| 義手・義足 | 身体の一部か？破壊されたら傷害罪か器物損壊罪か？ |
| 人工臓器 | 移植を受ける権利は「生存権」に含まれるか？ |
| ブレイン・コンピュータ・インターフェース（BCI） | 思考の読み取りは「通信傍受」に当たるか？ |
| サイボーグ化 | 人間の定義は変わるか？法的主体性に影響するか？ |
| 意識のアップロード | アップロードされた意識は「人間」か？権利を持つか？ |

### ニューロライツ（神経権利）（鈴木）

チリが2021年に世界で初めて「ニューロライツ」を憲法に組み込んだ：
1. **認知的自由**：自分の精神活動への介入を拒否する権利
2. **精神的プライバシー**：脳データの保護
3. **精神的完全性**：テクノロジーによる人格の改変からの保護
4. **公平なアルゴリズムへのアクセス**：脳関連AIの公正性

> これは「デジタル人権」のさらに先、「神経人権」というフロンティア。
> 身体の法的概念が「脳の中」にまで拡張されている。

### 人間の「定義」の法的危機（藤田）

- 遺伝子編集された人間は「自然人」か？
- サイボーグ化が進んだ身体の何%が「人工」なら、法的扱いが変わるか？
- 意識がデジタル化された存在の法的地位は？
- **「人間であること」の法的定義が、テクノロジーによって根底から問い直される**
- これはTACIEの「カーネルレベル」の問題——法体系の最深部に関わる

## 法のゲーミフィケーション——法的実験としてのゲーム

### シリアスゲームによる法的実験（藤田）

- **法的シミュレーションゲーム**：仮想社会で法律を設計し、その帰結を体験
- 参加者が「立法者」となり、法律を制定。AIが市民の行動をシミュレート
- 法の効果を「プレイ」することで、直感的に理解できる
- **法の「プロトタイピング」**：本番前にゲーム空間でテスト

### 具体的な設計案（サラ）

```
[ゲームフロー]
1. 社会的課題が提示される（例：AI失業問題）
2. プレイヤーチームが法案を起草
3. AIが100万人の仮想市民の行動をシミュレート
4. 法案の効果が可視化される（失業率、GDP、幸福度等）
5. 他チームの法案と比較・議論
6. 改善案を作成して再シミュレーション
7. 最終的な法案を投票で選択
```

### ゲーミフィケーションの先行事例（山本）

- **Democracy 4**（ゲーム）：政治シミュレーションゲームが法制度を疑似体験させる
- **MinecraftによるUN-Habitat**：都市計画をゲーム内で市民参加型に
- **Decidim**（バルセロナ）：市民参加プラットフォームにゲーム要素を統合
- **Foldit**（タンパク質折りたたみゲーム）：市民がゲームで科学研究に貢献 → 法制度設計にも応用可能

### ゲーミフィケーションの危険性（鈴木）

- 法を「ゲーム」として扱うことの倫理的問題
- 法は人の人生を左右する重大な制度 → 「楽しさ」で扱うことの不適切さ
- ゲーム化が参加のハードルを下げる一方で、議論の深さを犠牲にするリスク
- **「浅い参加」の大量生産 vs 「深い参加」の少数精鋭**

### 法のA/Bテスト（ハート）

- テクノロジー企業がUIのA/Bテストを行うように、法律もA/Bテストできるか
- 同じ問題に対する2つの法的ソリューションを異なる地域で試験運用
- 結果を比較して、より良い方を全国展開
- **法の下の平等との緊張**：同じ国民に異なるルールが適用される
- ただし特区制度（国家戦略特区等）は既にこれに近い
